## Installation
Muffin can be installed through conda. Right now it is only available on a private channel, but it is planned to migrate to conda-forge or bioconda. The extra channels are required for dependencies.
```sh
conda install -c pdelangen13 -c bioconda -c conda-forge muffin
```
